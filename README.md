# The Ethical path
<img width="1283" height="719" alt="mainmenu" src="https://github.com/user-attachments/assets/5d1b714e-9fee-42ce-ae62-d910feaec120" />

Navigate a series of complex scenarios in a modern workplace where every choice matters.
Your decisions shape not only your career, but also the ethical consequences of your actions.

- Interactive Storytelling: Explore multiple storylines through video-driven scenes.

- Meaningful Choices: Each option influences the outcome in subtle ways.

- Customizable Experience: Control music, skip scenes, and adjust pacing to suit your play style.

Face dilemmas, weigh consequences, and discover how your path unfolds.

# Installation and run

### To create a virtual environment:
```bash
python3 -m venv .
source ./venv/bin/activate
```

### Run
```bash
pip install -r requirements.txt
cd Ethic_game
python3 src/game.py 
```
